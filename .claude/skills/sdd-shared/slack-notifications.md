# SDD 알림 프로토콜 (개인용 — 비활성)

이 프로젝트는 혼자 쓰는 개인 프로젝트라 **외부 알림(Slack 등)을 사용하지 않습니다.**

다른 sdd-* 스킬이 "STARTED / MILESTONE / COMPLETED / FAILED 알림을 발송하라"고 이 문서를 참조하더라도,
**모든 알림 단계는 아무 동작 없이 건너뜁니다(no-op).** 알림용 스크립트 호출이나 외부 API 요청을 하지 마세요.

진행 상황은 일반 대화 출력으로만 사용자에게 보고합니다.

> 나중에 알림이 필요해지면, 이 문서를 알림 프로토콜로 교체하고 `.specify/`에 알림 설정 파일을 추가하면 됩니다.
